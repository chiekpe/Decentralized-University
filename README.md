# Decentralized University

A blockchain-based educational institution with tokenized degrees and student governance.

## Features
- Student registration and management
- Course creation and enrollment
- Tokenized degree system
- Governance token distribution
- Instructor verification

## Smart Contract
The main contract is located in `contracts/decentralized-university.clar`

## Technology Stack
- Clarity Smart Contract Language
- Stacks Blockchain
- Clarinet Development Environment
